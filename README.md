# 🖱 AutoClicker Pro Max

A simple and advanced AutoClicker built with Python and Tkinter.

---

## ⚙️ Features

- 🖱 Mouse auto click (left / right button)
- ⚡ Clicks Per Second (CPS) system
- 📊 Live click counter
- 🎛 Simple and clean GUI
- 🎮 Hotkeys support:
  - F6 → Start / Stop
  - F12 → Exit program
- 🎯 Preset CPS buttons (5 / 10 / 20 / 50)

## 🎮 How to use

1. Open the program
2. Choose mouse button (left or right)
3. Set CPS (Clicks Per Second)
   - Example: 10 CPS = 10 clicks per second
4. Press **Start (F6)** or click Start button
5. Press **Stop** to stop clicking

---

## 📊 CPS Examples

- 5 CPS → Slow clicking
- 10 CPS → Normal speed
- 20 CPS → Fast clicking
- 50 CPS → Very fast (may lag on some PCs)
